À propos :
------
  Tux Paint est un programme de dessin pour les jeunes enfants (environ 3 à
  10 ans).
  Il a été principalement développé pour satisfaire les besoins en logiciels
  éducatifs/ludiques du systéme d'exploitation Open Source « Linux », mais
  il est aussi compatible avec beaucoup d'autres plateformes telles que
   Windows, MacOS, BeOS, d'autres variantes Unix, etc.

  Ce paquetage est une collection de « tampons encreurs » pour l'outil
  « Tampons » du programme Tux Paint.


Installation :
-----------
  Linux/Unix:
  -----------
    En tant que superutilisateur 'root', tapez la commande :

      make install


    Par défaut, le répertoire "stamps" est placé en /usr/local/share/tuxpaint/

    Pour que le programme Tux Paint puisse voir les tampons, ils doivent être
    mis au même endroit que les autres données de Tux Paint.

    Voir la documentation concernant l'installation de Tux Paint, "INSTALL.txt",
    pour savoir comment changer la localisation des données.


  Windows:
  --------
    Le programme d'installation mettra les tampons à l'endroit approprié
    sur votre disque dur.
