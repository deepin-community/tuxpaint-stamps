2020.December.27
  * Build- and release-related improvements:
    + Small Android app to download stamps
      [Work in progress]
      Pere Pujal i Carabantes <pere@fornol.no-ip.org>

    + Windows Stamps installer localization definition.
      TOYAMA Shin-ichi <shin1@wmail.plala.or.jp>

    + Correctly linking License and Readme docs in
      Window Start menu.
      TOYAMA Shin-ichi <shin1@wmail.plala.or.jp>

    + Getting PO<->TXT scripts to sort output into the POT message catalog
      based on filenames (rather than the original text found in the "msgid"),
      to make it easier to work through translations on a per-category basis.
      (That is, things will appear less mixed-up, e.g., "A bassoon"
      (musical instrument), "A bat" (Halloween), "A bear" (mammals), etc.)

  * Localization updates:
    + Reordered _all_ PO files to record messages by stamp filename order
      (as now seen in the "tuxpaint-stamps.pot" message catalog),
      rather than by "msgid" (original English description -- which was via
      "msgmerge --sort-output ..." we were doing before; see above).

  * Build- and release-related improvements:
    + Packaging script for Android
      - Creates individual, mostly non-locale-specific ZIP files
        for each set of stamp categories (imagery, sound effects,
        and text descriptions), plus each spoken descriptive sounds
        on a per-stamp category, per-language/locale basis.
      - These files will then live on a server somewhere, and a
        small Android app. will be provided that can fetch whichever
        collections the user wants.  XML files defining the selection
        UI and available resource files are generated as part of this
        shell script.
      - by Pere Pujal i Carabantes <pere@fornol.no-ip.org>
      - clean-up & commit by Bill Kendrick <bill@newbreedsoftware.com>
