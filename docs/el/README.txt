README.txt για Tux Paint Stamps

Σχετικά:
--------
  Το Tux Paint είναι ένα πρόγραμμα σχεδίασης για μικρά παιδιά (ας πούμε,
  3-10 ετών). Το πρόγραμμα δημιουργείται για να καλύψει εκπαιδευτικές /
  ψυχαγωγικές ανάγκες, για το λειτουργικό σύστημα ανοιχτού λογισμικού
  "Λίνουξ", αλλά είναι συμβατό και με πολλές άλλες υπολογιστικές πλατφόρμες,
  που περιλαμβάνουν τα Windows, το MacOS, το  BeOS, άλλες παραλλαγές Unix,
  κλπ.

  Αυτό το πακέτο είναι μια συλλογή από στάμπες για το εργαλείο "Στάμπα"
  του Tux Paint.


Εγκατάσταση:
------------
  Για Λίνουξ/Unix:
  ---------------
    Ως  υπερχρήστης 'root', εκτελέστε την εντολή:

      make install

    
    Ο προκαθορισμένος κατάλογος με στάμπες (όπου εγκαθίστανται οι στάμπες)
    είναι /usr/local/share/tuxpaint/
    
    Για  να μπορεί το Tux Paint να δει τις στάμπες, αυτές πρέπει να
    τοποθετηθούν στην ίδια θέση με τα άλλα δεδομένα του Tux Paint.

    Βλ. τα συνοδευτικά έγγραφα οδηγιών του Tux Paint, "INSTALL.txt",
    για πληροφορίες σχετικά με αλλαγή του πού εγκαθίστανται τα δεδομένα
    του προγράμματος.


  Windows:
  --------
    Κάντε διπλό κλικ στο εκτελέσιμο πρόγραμμα εγκατάστασης του Tux Paint 
    Stamps (αρχείο .EXE file) και ακολουθείστε τις οδηγίες.

    Θα ερωτηθείτε πού επιθυμείτε να εγκατασταθούν οι στάμπες του Tux Paint.
    Η προκαθορισμένη θέση είναι βολική, αν υπάρχει ο σχετικός διαθέσιμος
    χώρος στο δίσκο. Διαφορετικά, διαλέξτε κάποιαν άλλη, πιο βολική θέση.

    Σε αυτό το σημείο, κάντε κλικ στο 'Install' και θα εγκατασταθούν οι
    στάμπες του Tux Paint!

    (Θυμηθείτε ότι μπορείτε να προσθέσετε και τις δικές σας! Βλ. τα 
    συνοδευτικά έγγραφα οδηγιών του Tux Paint για λεπτομέρειες.)
